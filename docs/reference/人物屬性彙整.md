# 人物屬性彙整（奇點世界）

> 人物屬性相關設定與細規**單一彙整**於此。涵蓋：架構方案與主管決斷（**A+B**：詞盤驅動＋雙軌）、基礎屬性項、出身與念紋加權、疊加公式、戰鬥結算維度、與詞盤彙整對照表共用處、與他系統對齊。與 [詞盤彙整](詞盤彙整.md) §九（竅穴／詞元／連線對照表）、[人物角色模板](人物角色模板.md)（實體欄位）對接。

**主管決斷**：**A+B**。詞盤驅動（竅穴→普通加成、詞元入竅→特殊加成、連線→額外加成、裝備疊加）＋雙軌（少量基礎屬性由出身×念紋加權，其餘由詞盤與裝備疊加）。基礎不隨升級膨脹，成長主線在詞盤與裝備。

---

## 一、架構總覽（A+B）

- **A. 詞盤驅動**：竅穴打通→普通加成、詞元入竅→特殊加成、連線→額外加成；裝備附魔／強化再疊加。全部由後端計算認證。
- **B. 雙軌（基礎＋詞盤）**：角色有**少量基礎屬性**（3 項以內），由**出身**與**精神念紋**以**加權制**共同決定（出身加權小、念紋加權大）；其餘由竅穴、連線、裝備疊加。基礎不隨升級膨脹，成長主線在詞盤與裝備。

**有效屬性** ＝ 基礎（出身×念紋 加權）＋ 竅穴普通加成 ＋ 詞元特殊加成 ＋ 連線加成 ＋ 裝備。

屬性需與**竅穴／連線、裝備＝工具、戰鬥全自動、後端唯一真理、玩家＝NPC** 對齊。

---

## 二、架構方案背景（組員建議與決斷）

組員曾提出多種方案（詞盤驅動型、雙軌型、經典六維、戰鬥／生活分離、純效果池等）；**主管擇定 A+B**，理由：本案已定有精神念紋，念紋決定與哪類詞元共振、天賦方向；雙軌的「少量基礎屬性」可與念紋對接，其餘由竅穴／入詞／連線與裝備疊加，兼顧天生差異與詞盤驅動。  
（其餘方案細節從略，以 A+B 為唯一落地架構。）

## 三、基礎屬性項（3 項以內）

| 屬性項 | 說明 | 戰鬥／生活 |
| --- | --- | --- |
| **體質** | 承傷、負重、耐性 | 戰鬥＋生活 |
| **氣脈** | 續戰、技能／行動消耗、回復 | 戰鬥 |
| **靈敏** | 命中、閃避、行動序 | 戰鬥 |

- 基礎項**僅此 3 項**；不另設「力／智／運」等，避免與詞盤、裝備重複計價。
- 基礎值**不隨等級成長**；成長來自竅穴打通、詞元入竅、連線、裝備（永序守衡）。
- **可由竅穴打通產生的普通加成**與上表對齊（詞盤彙整 §9.1）；**詞元入竅產生的特殊加成**依詞元屬性（火、冰、堅韌、疾速等）見詞盤彙整 §9.2。

## 四、出身與念紋皆採加權制（係數）

**出身**與**念紋**皆採**加權制**（係數），不採固定加算值：**出身加權小、念紋加權大**。基礎 3 項由「基準 × 出身係數 × 念紋係數」或後端等價公式得出，係數僅作用於基礎，不作用於竅穴／裝備。

### 4.1 出身加權（小）

- 出身類型（地域、門派、階層等）對應**體質／氣脈／靈敏**各一組係數，數值區間小（例：0.95～1.05）。
- 後端維護「出身 → 體質係數、氣脈係數、靈敏係數」對照表。

### 4.2 念紋加權（大）

念紋決定與哪類詞元「共振」、天賦方向；對基礎 3 項的**加權較大**（例：1.0～1.25 或依設計）。

| 念紋類型 | 說明 | 加權方向（例） |
| --- | --- | --- |
| **均衡** | 工藝契約型，走向平衡平穩 | 體質／氣脈／靈敏係數皆 1.0 |
| **剛體** | 偏承傷、耐性 | 體質係數大、其餘 1.0 |
| **氣長** | 偏續戰、回復 | 氣脈係數大、其餘 1.0 |
| **疾敏** | 偏命中、行動序 | 靈敏係數大、其餘 1.0 |

- 實際係數由後端或設計常數表定義；**念紋加權 > 出身加權**（影響幅度較大）。

### 4.3 基礎值計算（概念）

- **基礎有效值**（每項）＝ 基準（如 1 或由常數表） × 出身係數 × 念紋係數。
- 或等價公式：先算出身加權小幅調整，再乘念紋加權。後端擇一實作，維持「出身小、念紋大」即可。

**第一版可簡化**：固定一組數值或僅用念紋類型對應一組係數，出身可省略。

## 五、疊加公式（與詞盤彙整對照表共用）

有效屬性由以下**順序**疊加，後端唯一真理。

### 5.1 公式（概念）

```
有效屬性[體質] = 基礎體質（出身×念紋 加權） + Σ 竅穴普通加成(體質) + Σ 詞元特殊加成(體質類) + 連線加成(體質類) + 裝備加成(體質類)
有效屬性[氣脈] = 基礎氣脈（出身×念紋 加權） + Σ 竅穴普通加成(氣脈) + Σ 詞元特殊加成(氣脈類) + 連線加成(氣脈類) + 裝備加成(氣脈類)
有效屬性[靈敏] = 基礎靈敏（出身×念紋 加權） + Σ 竅穴普通加成(靈敏) + Σ 詞元特殊加成(靈敏類) + 連線加成(靈敏類) + 裝備加成(靈敏類)
```

- **竅穴普通加成**：依 [詞盤彙整](詞盤彙整.md) §9.1「竅穴打通→普通加成」對照表，按已打通竅穴累加。
- **詞元特殊加成**：依詞盤彙整 §9.2「詞元屬性→特殊加成」；若詞元屬性對應到體質／氣脈／靈敏相關（如堅韌→體質類、疾速→靈敏類），則累加入對應有效屬性。
- **連線加成**：依詞盤彙整 §9.3 連線模板，觸發的加成 ID 對應到體質／氣脈／靈敏或戰鬥維度時累加。
- **裝備**：裝備上的附魔／強化提供「工具維度」加成，與人物有效屬性相加後進入戰鬥結算。

### 5.2 戰鬥結算用維度（全部由有效屬性＋裝備推導）

不另設「攻擊力／防禦力」裸值；戰鬥時由下列推導：

| 維度 | 推導來源（例） |
| --- | --- |
| 傷害輸出 | 技能＋詞元屬性＋有效靈敏／氣脈（若設計有）＋裝備 |
| 承傷／減傷 | 有效體質＋堅韌類特殊加成＋裝備 |
| 命中／閃避 | 有效靈敏＋疾速／靈敏類詞元＋裝備 |
| 行動序／速度 | 有效氣脈、有效靈敏＋疾速類＋裝備 |

公式具體係數（如體質每點對應減傷多少）由後端常數或表驅動定義。

| 詞盤彙整章節 | 共用內容 |

## 七、與他系統的對齊

| 系統 | 對齊要點 |
|------|----------|
| **詞盤** | 普通加成＝竅穴打通；特殊加成＝詞元入竅；連線＝額外或套裝效果。全部由後端計算認證。 |
| **裝備＝工具** | 裝備提供「工具維度」加成（如鋒利→傷害、堅韌→承傷）；與人物屬性相加或加權後進入戰鬥結算。工具熟練度可視為「裝備維度」的係數或解鎖門檻。 |
| **戰鬥全自動** | 分秒輸贏＝f(有效屬性, 技能, 裝備, 隨機因子)；有效屬性＝基礎＋內盤加成＋裝備，無前端即時操作。 |
| **玩家＝NPC** | 同一套屬性架構與計算公式；NPC 差異可來自「基礎值＋內盤配置＋裝備」不同，支援萬名多差異個性。 |
| **永序守衡** | 屬性有來源（出身×念紋、竅穴、詞元、裝備），無「憑空加點」；成長＝擴充內盤與裝備，後端唯一真理。 |
