## 一、社會層級與玩法對應

| 層級 | 社會單位 | 可做的事 |
| --- | --- | --- |
| 低 | 種族、氏族 | 基礎安全、血緣／文化任務、氏族小據點 |
| 中 | 門派、聯盟 | 共鬥、資源共享、據點建造／祭煉／設施、門派任務 |
| 高 | 城鎮 | 經營（開店、市集、接任務）、行政保護、城鎮稅收與維護 |
| 巨型 | 國家、跨門派聯盟 | 國戰、資源爭奪、能量池穩定、跨區域規則 |

- 社會單位功能：優先名份與名單＋資源共享＋領地（建造）；歸屬＋集團身份對接。

## 二、真實經濟（收斂要點）

- **通膨控制**：產出有來源、回收成閉環（存在稅／維護費＋裝備損壞＋傳送付費＋據點維護）、可驗證。
- **玩家間交易**：
  - **P2P**：靠近他人，點擊彈窗內「給予」（金錢或物品）。
  - **市集／拍賣**：大城市專有交易所建築，系統或據點抽成（例 5%）進入閉環（參考阿爾比恩／EVE／Wakfu）。
  - UI 原則：經濟介面從彈窗或建築互動開啟，無獨立按鈕。
- **物價**：供需＋世界事件影響；首版可部分固定價。
- **產出—消耗**：產出—消耗鏈、房間／據點能量池、**禁止無中生有**。
- **無職業綁定**：裝備＝工具；工具熟練度；全身多部位裝備。

## 三、戰鬥與死亡（參考）

- **戰鬥全自動**：以文字 log 呈現；分秒輸贏由裝備、技能等後端數據與屬性配置決定（田忌賽馬）。
- **體驗重心**：戰前配置＋戰中讀 log＋戰後生死獎勵敘事；比較＝敘述與用詞，無數值試算。
- **死亡**（可依本專案決斷）：會死；不死光環復活。失敗剝光、裝備掉原地、重生居住點；祭煉有主裝備可召回。與 KEYWORDS_ASSESSMENT 死亡方案 A/B 等可對接。

## 四、自創技能（詞元—詞盤—敘事，僅作參考）

- 本專案已決：自創技能＝**組合既有模組**＋**工具熟練度延伸**（見 KEYWORDS_ASSESSMENT）；以下為他專案之擴充參考，非本專案定案。
- **有中生有**：工具解鎖初階；熟練度→進階／異變；技能組合。
- **詞元—單詞—詞盤—敘事—陣列**（可選）：玩家收集詞元→詞元組成單詞→詞盤祭煉成敘事→多敘事組陣列；**限制**＝系統判定中文邏輯、語順、語意，合則生成屬性加成或技能效果。
- **現實比喻**：詞元＝世界可收集原子；詞盤＝晶片；陣列＝高階產品；世界中有人專門祭煉空白詞盤。
- **強化例**：我會跑→speed+1.5％；技能例：我以火擊之→單體火傷，我以烈火連擊之→連擊或小範圍。
- **系統命名**：語意萃取簡名（2～4 字）；名稱由系統生成，不開放玩家自訂。描述寫完整敘事＋發明者＋效果。
