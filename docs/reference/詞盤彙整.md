# 詞盤系統彙整（奇點世界）

> 詞盤系統相關設定與細規**單一彙整**於此。涵蓋：主管設想（口述、用語）、本體論與世界觀、詞元分類、與個體／自創技能／裝備附魔的對接、邏輯閉環、**後端可實作細規**（竅穴／詞元對照表、語意邊界）、遊戲大數據檢視。鎂的產消閉環見 [經濟彙整](經濟彙整.md) §四。與 [人物屬性彙整](人物屬性彙整.md) 共用對照表。

**來源**：主管口述紀錄、組員腦洞補完（與現有架構閉環）、詞盤細規（後端可實作部分）。未經主管補充前，世界觀與用語不擅自詮釋或擴充。

---

## 一、主管設想（口述紀錄）

- **token**：無處不在卻無法直接接觸的存在，萬事萬物皆受它影響。人類因它而能將所見表達出來——紀錄→文字→詞元。**詞元可強化內、外**：內＝內盤、竅穴、人身；外＝裝備、外在詞盤。
- **token 與物質**：token 使物質**富態化、易質變**，與剝名、附魔、祭煉等「名與形質可變」同源。
- **作為貨幣**：其名稱被紀錄為「鎂」。**token 的作用**：賦權。
- **鎂字**：較為獨特的詞元；token 使人知悉所有物質的名。
- **詞盤系統的對接**：對接自創技能及個體、裝備附魔強化。
- **賦權與剝名**：因 token，人類能將「名」剝下；失去名之物消散；剝下的名帶有物質特有屬性，接附魔、強化、成長。**邊界**：同類指人與人，對人剝名 token 沒反應；人對獸、對物剝名可行（陽光奇點）。
- **內（詞）盤與竅穴**：詞盤在人身即內（詞）盤，以竅穴為詞元載體；連線後產生各種加成，由後端計算認證。**竅穴加成**：打通為普通加成；加入詞元為特殊加成。
- **設計原心**：收集字、造句。蒐集詞元→組成單詞／造句→**詞盤祭煉成敘事**→**系統判定**中文邏輯、語順、語意→合則生成屬性加成或技能效果；技能名**系統萃取簡名**（2～4 字）。

### 用語（依主管指定）

- **token**：玩家（含 NPC）滿世界蒐集的物件。**中文名稱**：幣。**又名**：詞元。**定位**：世界通用幣最小單位。**作為貨幣之名稱**：鎂。

## 二、詞盤的本體論：名、盤、登錄

- **盤**：「詞」來自詞元，「盤」為承載、排列詞元的器或總匯。世界萬物皆有「名」，名由詞元構成；**詞盤**即「名的登錄處」。
- **造句與祭煉**：收集詞元→組成單詞／造句→**詞盤祭煉成敘事**（登錄）→多敘事可組**陣列**。系統判定中文邏輯、語順、語意，合則生成屬性加成或技能效果。
- **三層詞盤**：  
  - **個人詞盤（內盤）**：個體與生俱來或後天蒐集；在人身即**內（詞）盤**，以**竅穴**為詞元載體。竅穴打通→普通加成；詞元入竅→特殊加成；**連線**→各類加成；均由後端計算認證。  
  - **機構詞盤**：各國國有金融機構內的「鎂字流轉總帳」——誰持多少鎂、誰對誰賦權、哪些契約被承認。  
  - **世界詞盤**：記載一切可名之物的總目錄；透過探索、傳承、交易，文明把新的「名」寫進世界詞盤邊緣。
- **名與物質的閉環**：未被詞元觸及的物質尚無「名」，無法被紀錄、交易。冒險者蒐集詞元或剝名，即擴充可被表達的邊界；鎂字為「名之名的單位」。
- **剝名**：人類將物或獸上的「名」剝下。**代價**：該物或獸消散。**所得**：剝下的名帶有屬性（火、堅韌、疾速、靈敏等），成為**帶屬性的詞元**，供附魔、強化、成長。人對人剝名 token 無反應；人對獸、對物可行。

---

## 三、詞元的分類與層級

- **鎂字**：世界通用貨幣與賦權基準；經機構賦權後流通；不從剝名取得。
- **剝名所得詞元**：從物或獸剝下的名，帶來源屬性；可入竅穴、寫入裝備、參與自創技能；依稀有度可視為地域級或更高。
- **通用詞元**：人人可感應、可納入內盤；對應常見物產名、基礎技藝名；入竅可產生基礎型特殊加成。
- **地域／傳承詞元**：特定地域、族群、師門才有；入竅或裝備觸發較強特殊加成或連線。
- **禁忌／失傳詞元**：曾從世界詞盤抹除或封存；本專案陽光向可設為稀少但不走殘酷代價。
- **個體專屬詞元**：極少數個體與某「名」共振產出、僅此人可用；與自創技能掛鉤。

---

## 四、詞盤與個體（深度對接）

- **內（詞）盤與竅穴**：詞元以**竅穴**為載體。**竅穴打通**→**普通加成**（體質、氣脈、基礎抗性等）；**詞元入竅**→**特殊加成**（依詞元屬性：火竅、疾速竅等）。多竅依規則形成**連線**，連線帶來額外加成；均由後端計算認證。
- **個體即「行走的詞盤」**：先打通竅穴得普通加成，再詞元入竅得特殊加成，透過連線放大；個體強度＝盤的深度與廣度＋連線加成。
- **精神念紋**：族裔可能帶有不同性質的念紋（偏戰鬥、工藝、契約等），決定與哪類詞元「共振」更容易，即**天賦方向**。
- **個體養成迴圈**：探索與戰鬥→剝名或蒐集詞元→以鎂交易或取得資源→打通竅穴→詞元入竅→連線→詞元排列成自創技能、剝下的名寫入裝備附魔／強化；後端統一計算並認證所有加成。

---

## 五、詞盤與自創技能（深度對接）

- **技能即「可被執行的名」**：一組詞元依特定順序與條件被喚起、賦權、執行。**自創技能**＝把內盤上的詞元（含剝下的名）排列成世界尚未登錄的新名→造句→**詞盤祭煉成敘事**→系統判定語意通過即生成效果。
- **自創技能的誕生**：內盤具備足夠且相容的詞元時，造句／排列成敘事→**祭煉**於詞盤→**系統判定**中文邏輯、語順、語意，合則生成屬性加成或技能效果；消耗剝下的名、鎂或兩者。
- **系統命名**：技能名稱由**系統萃取簡名**（2～4 字），不開放玩家自訂；描述＝完整敘事＋發明者＋效果。
- **登錄與流通**：自創技能可僅止於內盤（獨門），或申請登錄到工會、門派或世界詞盤邊緣；多敘事可組**陣列**，對應高階效果。

---

## 六、詞盤與裝備附魔強化（深度對接）

- **附魔的根源＝剝下的名**：從物或獸剝下的「名」帶有該來源的屬性；**附魔**即把帶屬性的詞元透過儀式、鎂與念紋寫入裝備的詞盤（附魔槽），使裝備獲得對應權能（鋒利、堅韌、火炎、疾速等）。
- **附魔與強化**：裝備即「外在詞盤」，具有限槽位。鑲嵌剝下的名＝一次賦權；**強化**＝以更多鎂或同名／同系詞元做二次賦權；**強化階**對應賦權深度；過載可設為可修復（陽光向）。
- **與內盤的共鳴**：裝備附魔詞元與穿戴者內盤（竅穴詞元、念紋）相容時可觸發**共鳴**（加成或減耗）；不相容則可能折損。

---

## 七、賦權的兩面

- **賦權的兩面**：一為「賦予名分」——使某物、某約、某筆帳被承認、可紀錄；一為「賦予權能」——持鎂者能交換、訂約、執行技能、驅動附魔。鎂字經機構賦權後流通；各國共用同一套鎂。

---

## 八、邏輯閉環與設計邊界

- **世界觀閉環**：token 無處不在→名由詞元構成→人類能剝名（僅物與獸）→名／詞元可流轉入內盤（竅穴、連線）、裝備（附魔槽）、技能（造句→祭煉→系統判定→授名）→鎂字經機構賦權通用→表達、交易、賦權、成長皆在「名」的體系內完成。
- **資源與玩法閉環**：探索／戰鬥→剝名或蒐集詞元→詞元入內盤（竅穴→入詞→連線）→造句／祭煉成敘事→系統判定語意→自創技能或陣列；剝下的名可寫入裝備附魔與強化→鎂經機構賦權用於交易、強化、技能消耗→個體與裝備變強→再探索。
- **設計邊界（陽光奇點）**：人對人剝名 token 無反應；禁忌詞元可稀少但不走殘酷代價；附魔過載可設為可修復。
- **閉環自檢**：詞元來源＝蒐集＋剝名；詞元去向＝內盤（竅穴／連線）＋造句／祭煉（技能／陣列）＋裝備附魔／強化；鎂＝經機構賦權、通用媒介（產消見經濟彙整）。

## 九、後端可實作細規

與 [人物屬性彙整](人物屬性彙整.md) 共用對照表。後端依此計算「竅穴打通→普通加成」「詞元入竅→特殊加成」「連線→額外加成」。

### 9.1 竅穴打通 → 普通加成（§2.1）

| 竅穴類型（可擴充） | 打通後普通加成項 | 數值／效果說明（例） |
| --- | --- | --- |
| 體竅 | 體質 | 承傷減免、負重、耐性；數值＋N 或百分比 |
| 氣竅 | 氣脈 | 續戰、技能／行動消耗、回復；數值＋N 或百分比 |
| 敏竅 | 靈敏 | 命中、閃避、行動序；數值＋N 或百分比 |

- **竅穴總數／上限**：由後端常數或角色模板定義（例：每人 8～12 竅）。**打通條件**：消耗鎂、任務或達到某條件，由專案再定。

### 9.2 詞元屬性 → 特殊加成（§2.2，詞元入竅後）

| 詞元屬性（可擴充） | 特殊加成效果（例） |
| --- | --- |
| 火 | 火系傷害＋、火抗＋、火系技能增幅 |
| 冰 | 冰系傷害＋、冰抗＋、遲緩／凍結相關 |
| 堅韌 | 物防＋、破防抗性 |
| 疾速 | 行動序、跑速、攻速＋ |
| 靈敏 | 命中、閃避＋（若未併入普通） |
| 記憶 | 技能冷卻、敘事相關（可選） |

- **詞元屬性表**：後端維護「允許的詞元屬性」清單；剝名／蒐集得到的詞元帶其中一類或多類標籤。**同竅多詞元**：規則需另定（僅取最高或疊加上限）。

### 9.3 連線加成規則（§2.3）

- **連線**：多個詞元竅穴依規則（經脈拓樸、星圖等）形成連線；連線成立時觸發**額外加成**。
- **規則例**：同一「屬性系」的竅穴達 N 個以上連通→觸發該系套裝加成（如火系 3 竅連線→火傷＋5％）。
- **後端**：可定義「連線模板」表（哪些竅穴編號相連、觸發何種加成 ID），戰鬥／結算時查表彙總。

### 9.4 系統判定語意邊界（造句／祭煉）

- **輸入**：玩家／NPC 提交的「造句」＝一串詞元或一句敘事（如「我以火擊之」）。
- **輸出**：通過→技能名（系統萃取 2～4 字）、效果類型、數值或係數；不通過→不生成技能、可回傳原因（選用）。
- **邊界**：僅接受**允許的詞元集合**內之組合；語順、語意由規則表或關鍵字匹配決定，不開放任意自然語言。
- **敘事範本（例）**：`[己][動詞][目標][屬性]`；後端維護「可參與造句的詞元 ID 表」及「詞元→語意標籤」對照。同一造句僅對應一組技能名與效果；重複提交不重複授名（已登錄則回傳已有技能名）。
- **系統授名**：技能名由後端萃取簡名，不開放玩家自訂；描述＝完整敘事＋發明者＋效果。

### 9.5 與人物屬性彙整的對接

- **普通加成項**：§9.1 為竅穴打通帶來的**成長型**普通加成；人物屬性的「基礎」為念紋／出身帶來的**天生**少量值，兩者相加後再與 §9.2 特殊加成、§9.3 連線、裝備疊加。
- **戰鬥結算**：有效屬性 ＝ 基礎（出身×念紋 加權）＋ 竅穴普通加成 ＋ 詞元特殊加成 ＋ 連線加成 ＋ 裝備；後端依此順序彙總後代入戰鬥公式。

## 十、擴充用語表

| 用語 | 說明 |
|------|------|
| 收集字／造句 | 收集詞元→組成單詞／造句→詞盤祭煉成敘事→多敘事組陣列；造句＝把字排列成有意義的結果（技能、附魔、連線、登錄名等）。 |
| 祭煉／敘事／陣列 | 詞盤祭煉成敘事＝將造句登錄於詞盤，系統判定語意後生成效果；多敘事可組陣列。 |
| 系統命名 | 技能名稱由系統萃取簡名（2～4 字），不開放玩家自訂；描述＝完整敘事＋發明者＋效果。 |
| 內（詞）盤 | 詞盤在人身之稱；以竅穴為詞元載體，打通／入詞／連線產生加成，由後端計算認證。 |
| 竅穴 | 人身上承載詞元的載體／落點；打通後普通加成，詞元納入後特殊加成並可參與連線。 |
| 竅穴打通 | 竅穴開啟，產生普通加成（體質、氣脈、基礎抗性等）。 |
| 詞元入竅 | 將詞元納入已打通之竅穴，產生特殊加成（依詞元屬性）；多竅連線後加成由後端認證。 |
| 普通加成 | 僅打通竅穴即得之加成，不依詞元種類。 |
| 特殊加成 | 詞元入竅後依該詞元屬性產生之加成；連線可再放大。 |
| 詞元竅穴／連線 | 多個詞元竅穴依規則連線後產生各類加成；後端計算認證。 |
| 詞盤容量 | 個體內盤或裝備可承載的詞元上限，可隨成長或品質擴充。 |
| 念紋 | 先天或後天與某類詞元共振的印記，決定天賦方向。 |
| 剝名 | 因 token 賦權，將物或獸上的「名」剝下；物／獸消散，剝下的名帶屬性，供附魔、強化、成長。人對人剝名 token 無反應。 |
| 登錄 | 把名（技能、契約、所有權）寫入機構或世界詞盤，使之可流通或可查。 |
| 附魔槽／詞元槽 | 裝備上可鑲嵌詞元的位置，即該裝備詞盤容量。 |
| 強化階 | 同一附魔名被重複賦權的深度，對應附魔強度。 |
| 共鳴 | 裝備附魔詞元與穿戴者內盤（竅穴詞元、念紋）相容時的加成或減耗。 |

## 十一、遊戲大數據檢視（與整體設計對齊）

| 維度 | 整體設計要點 | 詞盤對齊狀況 |
|------|--------------|--------------|
| **經濟** | 單一通用貨幣＋產消閉環 | **鎂**＝世界通用貨幣、經機構賦權；產消閉環見 [經濟彙整](經濟彙整.md) §四。 |
| **自創技能** | 組合模組＋後端驗證＋工具熟練度延伸 | **造句→祭煉→系統判定→系統授名**＝詞元排列＋後端驗證＋系統萃取 2～4 字；工具熟練度可疊加於竅穴／連線或裝備。 |
| **裝備** | 裝備＝工具；全身多部位；工具熟練度 | 裝備＝**外在詞盤**；附魔槽、強化階、共鳴與「工具」語義一致；剝下的名寫入裝備＝權能來源有據，不可無中生有。 |
| **不可無中生有／永序守衡** | 不可僅憑文字創造物件 | 詞元來源＝蒐集＋剝名（剝名則物／獸消散）；技能＝造句＋系統判定通過才授名；鎂由機構賦權、不從剝名取得。 |
| **後端唯一真理** | 數據、量化後端計算 | 竅穴打通／入詞／連線、祭煉、系統判定語意、系統授名、附魔／強化階，**均由後端計算認證**。 |
| **玩家＝NPC** | 同一套規則與身份 | token＝玩家（含 NPC）蒐集的物件；詞元、內盤、剝名、造句對 NPC 同一套邏輯。 |
| **社會／門派／城鎮** | 門派、氏族、城鎮、國家 | 登錄可寫入工會、門派、世界詞盤；機構詞盤由各國國有金融機構執掌。 |
| **戰鬥** | 全自動、文字 log；輸贏靠裝備與技能 | 技能＝可執行的名；強度與詞元組合、竅穴連線、裝備附魔／共鳴掛鉤；戰鬥結果由後端依詞盤與裝備數據計算。 |

**結論**：詞盤系統與現行關鍵字及評估決斷對齊良好。細規已涵蓋竅穴／詞元對照表與語意邊界；鎂的產消見經濟彙整；可選補齊 NPC 詞盤／蒐集行為與觀測坍縮介面。
