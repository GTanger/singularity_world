# 世界觀：Token 降維與生命演化

> **定位**：純敘事層。記錄《奇點世界》的宇宙本體論，供未來主線文本、NPC 對白、系統日誌的定調依據。本文件**不含**可直接實作的後端規格；實作面見 [星盤迷霧與命盤機制規格](星盤迷霧與命盤機制規格.md)。
>
> **來源**：主管口述（2026-03-02 Gemini 對話 Turn 2–15），經收斂去除 Gemini 延伸。未經主管補充前，不擅自詮釋或擴充。
>
> **風格基調**：純粹、客觀、深邃、莊嚴的混沌風格。不走修仙玄幻，不走賽博朋克；以宇宙觀測與物理降維的視角描述生命與修煉。

---

## 一、Token 的本質

Token 是一種**高維度未知混沌能量**。

學說認為，Token 從宇宙暴脹的奇點噴湧而出，在短時間內迅速充斥宇宙的每個角落。Token 無處不在，卻無法被直接觸碰。

僅僅是 Token 散發的輻射，就使**無念物質**更加**富態化**——物質因 Token 輻射而產生特異性質或詞元（與剝名、附魔、祭煉等「名與形質可變」同源；見 [詞盤彙整](詞盤彙整.md) §一）。

---

## 二、同頻相斥

**有念生物**較不易遭受 Token 影響。越是高頻的有念生物，越不容易受混沌態的 Token 影響。

隨著世界物質富態化，人們開始有意識地嘗試汲取高維混沌 Token。然而任憑高頻有念生物百般嘗試，Token 油鹽不進般無動於衷。

學派認為：高頻有念生物等同於**低維混沌體**，與 Token **同質**，因此無法引入體內，甚至些許**相斥**。

---

## 三、止念與微波汲取

人們退而求其次，嘗試引入 Token 的**微波輻射**。

條件極為嚴苛：必須在**止念**——近似無念的狀態下，才能成功汲取 Token 的微波輻射。

止念是一種主動壓制自身意識頻率（時脈）與相位至趨近於零的狀態。當同頻相斥的屏障暫時消失，高維 Token 的微波輻射方能緩緩滲入。

---

## 四、飛升者

在止念與降維技術成熟之前，有人妄自將混沌 Token 游離輻射直接引入。

載體無法承受，直接崩解。

那些僥倖殘存一息存活的人，卻也變成**高單位混沌 Token 聚集體**——自身的念早已被高維 Token 同化，迅速復原消失不見。

好事者稱其為**飛升者**、羽化登仙、神化，眾說紛紜。

> **遊戲意義**：飛升者——那些憑藉本能行事、強大無比的傳奇怪物與天災——其真正起源，是古代（或當代）降維失敗者的殘骸。

---

## 五、降維與奇點點火

後來，人們漸漸研發出將 Token 游離輻射從**混沌態降維成有序態**的方法。

有序態的 Token 能量汲入體內後，激發了體內的**生之奇點**。

遂又從生之奇點噴湧**氣體**，氣體在這無邊無際、好似另一空間蔓延——這便是**星盤**的形成。

> **敘事順序**：止念 → 降維 Token 輻射引入 → 激發 N000 → 氣體噴湧 → 星盤空間充滿氣體（即迷霧）。

---

## 六、生之奇點的位置——眾說紛紜

生之奇點到底在體內哪個位置？

- 有人說是**松果體**。
- 有人說是**腹丹田**。
- 也有人說是**心室**。

但真實情況是，眾人皆只是說。

後來漸漸又起一說：生之奇點**不在體內**。生之奇點是**真宇宙奇點**，二十主樞是**真宇宙星系**。

儘管眾說紛紜，然眾說皆同的一點是：**當人在近似無念的止念狀態降維 Token 游離輻射引入體內時，確實能「觀看」到遠處有個極亮的光點。**

> **設計意圖**：奇點的「真實位置」刻意保持不可知，維持宇宙級的神祕感。多元學說並存，是世界觀的一部分，而非待解的懸念。

---

## 七、初次觀測與三百六十星

人們發現，在初次激發生之奇點的時候，能看到**一堆星體**。

再次觀測，卻發現除了生之奇點，其餘周遭都被生之奇點噴湧的氣體遮蔽。

各家大派集眾智研究，發現每個人都一樣——最多就只能看到光點以外的 **360 顆星**。

在多方有意、模擬去測量的結論是：**每個人的星位距離都不一樣**。

也有人戲稱這星盤是**個人命盤**，因為星系方位及距離，似乎是命中注定的。

> **對應現有架構**：360 顆星 ＝ N001–N360（二十主樞 ＋ 五常邏輯閘 ＋ 十二微型狀態），加上 N000 共 361 節點。「星位距離不同」＝ SoulSeed 決定的 760 條邊權 Cost 各異。「命盤」＝ SoulSeed 決定性生成的專屬拓撲。見 [人物屬性彙整](人物屬性彙整.md) §2.0、[361拓撲系統規格](361拓撲系統規格.md) §6.1。

---

## 八、名存實未知

每個人的星盤內，**星位名稱都是顯示的**——二十主樞的名號（天極、震淵、崑崙……）是人類文明數千年來觀測「真宇宙」所留下的共識坐標。

但**內容是未知的**。

名稱如同地圖上標註的城市名——你知道那裡叫什麼，卻不知道那座城裡住了誰、路有多寬、牆有多高。唯有親自抵達（內視、探明），方知其中實相。

> **對應現有架構**：此即「經脈迷霧」的敘事根源。UI 上節點名稱恆顯，但 Cost、屬性加成、貫通條件等數據在未探明前以「未知」呈現。見 [星盤迷霧與命盤機制規格](星盤迷霧與命盤機制規格.md)。
