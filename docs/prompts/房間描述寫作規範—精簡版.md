# 房間描述寫作規範（精簡版）

**一句總結**：只寫可被五感或結構驗證的內容；禁止物性、物理感、秩序感、XX感、以及「呈現出／展現出／體現出」＋主觀結論；主觀感受留給玩家閱讀後自行產生。

---

## 原則

- 可寫：所見（光、色、材質、擺設）、所聞（氣味、聲音）、所觸（溫濕、地面）、空間結構（深處、盡頭、隔絕）。
- 不可寫：替玩家下「這裡給人什麼感覺」的結論；任何空洞的主觀評語。

---

## 禁止用詞（生成時一律不用）

| 類型 | 禁止 |
|------|------|
| 物性 | 物性穩固／堅實／穩定／最強／終點／最終回歸／呈現高度秩序；建築物性最強／最穩固的終點；精密物性結合、文明秩序與精密物性 |
| 物理＋感 | 物理穩固感／觸感／質感／死寂／安寧／重壓感／重量感；建築的物理心臟；呈現／展現絕對物理穩固感 |
| 秩序 | 秩序感極強／高／極端；高度／絕對秩序感；絕對秩序美；冷酷秩序感 |
| XX感 | 環境感（安定／安靜／肅穆／濕潤／壓迫感／飽和／精密…）；視覺感（飽和／清晰／深邃…）；空氣感（凝滯／乾燥…）；空間感（沈重／飽滿／密實…）；靜謐放鬆感、神聖寂靜感、物質壓迫感、轉折感、結束感、凝滯感 |
| 呈現／展現／體現 | ＋ 任何「…感」「…秩序」「物性」「物理」結論（例：呈現穩定安寧感、展現地下財富與秩序、體現物質過度補完） |
| 其它 | 物質過度補完（當氛圍結論）；透著一股孤寂感；質感極端飽和；分形幾何感→改「分形幾何」 |

---

## 改寫對照（用這個代替）

- 位置：用「建築最深處／最內側／生活區盡頭」，不用「物性終點／最穩固的終點」。
- 安靜：用「室內極靜／無雜音／厚牆吸音」，不用「環境感安定／呈現穩定安寧感」。
- 光線：用「光線飽和／光線呈琥珀色」，不用「視覺感飽和／呈現飽滿色調」。
- 空氣：用「空氣凝滯／空氣濕潤」，不用「空氣感凝滯／呼吸感飽滿」。
- 觸感：用「觸感」「材質厚實」「表面冰冷平滑」，不用「物理觸感／物理質感」。
- 陳設：用「陳設整齊／線條規整」，不用「秩序感／高度秩序」。

---

**自檢**：句子里有沒有「物性」「物理…感」「秩序感」「環境感」「視覺感」「空氣感」「空間感」或「呈現／展現／體現＋主觀詞」？有就改寫或刪掉。
