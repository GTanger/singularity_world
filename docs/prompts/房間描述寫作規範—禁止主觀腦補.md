# 房間描述寫作規範：禁止主觀腦補

**用途**：餵給 Gemini（或其它模型）作為生成「房間 description」時的硬性約束。  
**原則**：主觀感受應由**玩家閱讀具體文字後自行在腦海產生**，而非由 AI 在文案裡直接下結論（「這裡很有秩序感」「物性穩固」等）。禁止機械式、空洞的主觀評語。

---

## 一、核心原則

- 只寫**可被五感或具體狀態驗證**的內容：所見（光、色、材質、擺設）、所聞（氣味、聲音）、所觸（溫度、濕度、地面材質）、空間結構（深處、盡頭、隔絕）。
- **禁止**替玩家下「這裡給人什麼感覺」的結論；禁止「物性」「物理」「秩序」「XX感」等過度抽象或主觀的用詞。
- 若需要表達「很安靜」，寫「室內極靜」「無雜音」「厚牆吸音」等具體狀態，不要寫「環境感安定」「呈現穩定安寧感」。

---

## 二、禁止用詞與句型（清單）

### 2.1 禁止使用「物性」相關

- 禁止：物性穩固、物性堅實、物性穩定、物性極端堅固、物性最強、物性終點、物性最終回歸、物性呈現高度秩序、建築物性最強／最穩固的終點、物性與自然能量交匯終點、精密物性結合、文明秩序與精密物性結合、生活區物性終點、地下秩序精密物性、高度精確物性。
- 改寫方式：若指「位置」→ 用「建築最深處」「建築最內側」「生活區盡頭」。若指「結構厚實」→ 直接寫「牆體厚實」「結構紮實」，或刪除該句。

### 2.2 禁止「物理」＋主觀感

- 禁止：物理穩固感、物理觸感、物理質感、物理死寂、物理安寧、物理重壓感、物理重量感、物理邊界（若用於抽象「邊界感」）、建築的物理心臟、呈現／展現絕對物理穩固感。
- 改寫方式：「物理觸感」→「觸感」或「表面」；「物理質感」→「質感」或「材質」；「物理死寂／安寧」→「極靜」；「建築的物理心臟」→「建築核心」。

### 2.3 禁止「秩序／秩序感」當主觀評語

- 禁止：秩序感極強、秩序感高、高度秩序感、絕對秩序感、絕對秩序美、冷峻且整齊的秩序感、與外部一致的秩序感、在這種高度秩序中終結、體現／呈現高度秩序感、冷酷秩序感、秩序感極端。
- 改寫方式：若指陳設→「陳設整齊」「陳設規整」「線條規整」。其餘空洞秩序評語→ 刪除。

### 2.4 禁止「環境感／視覺感／空氣感／空間感」等抽象「XX感」

- 禁止：環境感安定／安靜／肅穆／濕潤／充滿張力／穩重有序／沉重具壓迫感／乾燥有序／精密且秩序井然／溫暖乾燥／極具工業精密度／從商業嘈雜轉向私人安寧／瞬間柔軟／極度飽和／硬朗且有序；視覺感飽和／清晰／深邃／溫潤飽滿／極度飽和／從…轉向…；空氣感凝滯／極度乾燥／溫暖濕潤／重新輕盈乾燥；空間感因能量密度而沈重／因物質富態化而飽滿／因光能富態化飽和而極其密實／因物件密集而飽滿；空間重量感、重量感隨…、踩踏感虛浮／鬆軟（若僅用來表達主觀感受）。
- 改寫方式：「環境感安定」→「室內安靜」；「視覺感飽和」→「光線飽和」；「空氣感凝滯」→「空氣凝滯」；「踩踏感虛浮」→「地面鬆軟」。其餘僅在描述「感覺」而無具體資訊者→ 改為可驗證狀態或刪除。

### 2.5 禁止「呈現出／展現出／體現出」＋主觀結論

- 禁止：呈現出／展現出一種…感、呈現出物性／物理性的最終回歸、呈現絕對物理穩固感／安寧／靜謐／秩序感／穩定感／平穩感／繁榮感／神聖寂靜感／放鬆感、呈現物質過度補完的安寧感／厚實感、呈現高度物質飽和感／文明化個人空間感／發達物質穩固感、展現絕對物理穩固／安寧、展現地下財富與算力／數字秩序的結合、體現物質過度補完的重壓／秩序、體現高度文明化的物質飽和／民生遮蔽／個人空間、體現極致的物理隱蔽性、呈現與外部精密機械感相反的溫馨氛圍、呈現溫和結束感、呈現從功能區回歸主幹的轉折感、呈現沈穩且凝滯感。
- 改寫方式：若後面接的是「光線／色調」等可觀察事實→ 可保留「呈現」並改為具體描述（如「光線呈琥珀色」）。若後面接的是感受、秩序、物性等→ 改為具體狀態描述或刪除。

### 2.6 禁止其它空洞主觀用語

- 禁止：物質壓迫感、物質過度補完（當作氛圍結論）、透著一股孤寂感、靜謐放鬆感、神聖放鬆感、質感極端飽和、光學呈現詭異華麗感、分形幾何感（→ 改「分形幾何」即可）。
- 建築位置：不用「物性終點」「物性最穩固的終點」，改用「建築最深處」「建築最內側」。

---

## 三、建議寫法（替代方向）

| 避免 | 改為（或刪除） |
|------|----------------|
| 物性穩固／堅實／最強終點 | 建築最深處／最內側；或刪除 |
| 物理穩固感／物理安寧 | 極靜；牆厚隔音 |
| 秩序感極強、高度秩序 | 陳設整齊、線條規整 |
| 環境感安定、呈現穩定安寧感 | 室內極靜、無雜音 |
| 視覺感飽和、呈現飽滿深琥珀色調 | 光線飽和、光線為深琥珀色 |
| 空氣感凝滯、呼吸感飽滿 | 空氣凝滯、空氣濕潤 |
| 展現出一種封閉且穩固的民居質感 | 外觀封閉穩固、門牆厚實 |
| 物質壓迫感、物質過度補完的重壓 | 刪除；或改為熱輻射強、空氣密度高 |
| 踩踏感虛浮 | 地面鬆軟 |
| 建築的物理心臟 | 建築核心 |

---

## 四、可保留的用法

- **呈現**：僅用於可驗證事實，例如「光線呈現琥珀色」「牆面呈現油脂光澤」。
- **質感／觸感**：用於具體材質時可保留，例如「材質厚實」「表面冰冷平滑」。
- **邊界**：用於具體視覺或空間邊界時可保留；避免「物理邊界」當作抽象感受。
- 世界觀用詞（若專案有定義）：例如「物質富態化」可保留為設定用語，但**不要**接「…感」「…秩序」「物性」等主觀結論。

---

## 五、檢查自問（生成後自檢）

1. 是否有「物性」「物理穩固感／物理安寧」「秩序感」「環境感」「視覺感」「空氣感」「空間感」等詞？有則改寫或刪除。
2. 是否有「呈現出／展現出／體現出」＋「…感」「…秩序」「物性」「物理」？有則改為具體描述或刪除。
3. 是否有「物質壓迫感」「物質過度補完」當作氛圍結論？有則刪除或改為可觀察狀態。
4. 每句是否都能對應到「看得到、聽得到、聞得到、摸得到或空間結構」？若僅是「給人某種感覺」的結論，則改寫或刪除。

---

**總結**：描述只寫**可被感官或結構驗證的內容**；**禁止**物性、物理感、秩序感、XX感、呈現／展現／體現＋主觀結論。主觀感受留給玩家在閱讀後自行產生。
