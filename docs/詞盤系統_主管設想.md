# 詞盤系統 — 主管設想（紀錄）

以下為主管口述之詞盤系統設想，由組員紀錄。未經主管補充前，不擅自詮釋或擴充。

---

## 主管設想（口述紀錄）

- **token 是什麼**：無處不在的卻又無法直接接觸的存在，萬事萬物皆受它影響。
- **token 使人類獲得詞元**：人類因為它，而能將眼睛所看到的表達出來。如何表達？紀錄。如何紀錄？文字。文字怎麼來？詞元。——**詞元可強化內、外**：內＝內盤、竅穴、人身；外＝裝備、外在詞盤。強化皆來自詞元的取得與運用。
- **token 與物質（世界設定補充）**：token 使得物質**富態化、易質變**——物質在 token 影響下更易呈現多態、更易發生質的轉變；與剝名、附魔、祭煉等「名與形質可變」的設定同源。
- **最直白的說法**：人類因為它，而能將眼睛所看到的表達出來。如何表達？紀錄。如何紀錄？文字。文字怎麼來？詞元。
- **作為貨幣**：它的名稱被紀錄為「鎂」。
- **token 的作用**：賦權。
- **瑞式族人**：世界皆知瑞式族人只要被家族檢測出天生精神念紋適合鎂字賦權，成人禮後就會被安排至各國國有金融機構專職，無他，鎂字僅適合瑞式族人血脈，故瑞式族人永遠保持中立。
- **鎂字**：較為獨特的詞元。token 使人知悉所有物質的名。
- **詞盤系統的對接**：對接自創技能及個體、裝備附魔強化。
- **賦權的補充**：因 token 的影響，人類能將「名」剝下。代價是失去名之物消散。剝下的名帶有些許物質特有的屬性。後續接附魔、強化、成長等系統。
- **對同類剝名**：同類指人與人。設定為 token 沒反應——人無法對人剝名。人對獸剝名可行。本專案為陽光奇點，不走黑暗設計風。
- **內（詞）盤與竅穴**：詞盤在人身稱為內（詞）盤，使用竅穴為詞元載體。連線後的詞元竅穴產生各種不同的加成，可由後端計算認證。
- **竅穴加成**：一般竅穴打通可能只是普通加成；加入詞元則會產生特殊加成。
- **設計原心**：收集字，然後造句。玩家蒐集詞元（字）→ 組成單詞／造句 → **詞盤祭煉成敘事** → 經**系統判定**中文邏輯、語順、語意，合則生成屬性加成或技能效果；可授予技能名稱（系統萃取簡名 2～4 字）。例：收集「我以火擊之」→ 詞盤登錄／祭煉成敘事 → 系統認證 → 授予「火球術」（單體火傷）；「我以烈火連擊之」→ 連擊或小範圍；「我會跑」→ speed+1.5％ 等強化。

## 用語（依主管指定）

- **token**：玩家（包含 NPC）滿世界蒐集的物件。**中文名稱**：幣。**又名**：詞元。**定位**：世界通用幣最小單位。**作為貨幣之名稱**：鎂。

## 組員腦洞補完（與現有架構閉環）

以下為組員在主管既定架構內之幻想補完，僅供主管揀選或否決，不視為正式設定。目標：龐大、豐富、深度，並與**自創技能、個體、裝備附魔強化**完整對接；**另案詞盤設計（RUST 專案）已融入本案**，不再另列區塊。

- **設計核心（本案融合）**：**收集字、造句**。流程：蒐集詞元 → 組成單詞／造句 → **詞盤祭煉成敘事**（登錄於內盤或世界詞盤）→ 多敘事可組**陣列**。系統判定中文邏輯、語順、語意，合則生成屬性加成或技能效果；技能名由**系統萃取簡名**（2～4 字），不開放玩家自訂；描述＝完整敘事＋發明者＋效果。比喻：詞元＝可收集原子，詞盤＝晶片，陣列＝高階產品；世界中有人祭煉空白詞盤。本專案另疊加剝名、內盤／竅穴、鎂、瑞式族人、附魔強化等世界觀。

### 一、詞盤的本體論：名、盤、登錄

- **盤**：若「詞」來自詞元，則「盤」可視為承載、排列詞元的器或總匯。世界萬物皆有「名」，名由詞元構成；**詞盤**即「名的登錄處」——哪裡有哪些物質已被知悉、被紀錄，就在詞盤上有位子。比喻：詞元＝可收集原子，詞盤＝晶片，多敘事組**陣列**＝高階產品；世界中有人專門**祭煉**空白詞盤。
- **造句與祭煉**：收集詞元 → 組成單詞／造句 → **詞盤祭煉成敘事**（登錄）→ 多敘事可組陣列。系統判定中文邏輯、語順、語意，合則生成屬性加成或技能效果。
- **三層詞盤**：  
  - **個人詞盤（內盤）**：每個個體與生俱來或後天蒐集而成的「小詞盤」，在人身即**內（詞）盤**，以**竅穴**為詞元載體。**竅穴打通**得普通加成，**詞元入竅**得特殊加成，**連線**後產生各類加成，均由後端計算認證。盤上詞元的種類與數量、以及竅穴連線，直接綁定個體的認知邊界與可操作範圍。  
  - **機構詞盤**：各國國有金融機構內由瑞式族人執掌的「鎂字流轉總帳」——誰持多少鎂、誰對誰賦權、哪些契約被承認，全在盤上。  
  - **世界詞盤**：記載一切可名之物的總目錄；無人能獨占，但透過探索、傳承、交易，文明不斷把新的「名」寫進世界詞盤的邊緣。
- **名與物質的閉環**：token 使人知悉所有物質的名。未被詞元觸及的物質就還沒有「名」，無法被紀錄、交易、納入文明。冒險者蒐集詞元或剝名，即擴充**可被表達的邊界**；鎂字被約定為「名之名的單位」，是衡量與賦權的基準。
- **剝名（賦權的具體操作）**：因 token 的影響，人類能將**物或獸**上的「名」剝下。**代價**：失去名之物或獸消散。**所得**：剝下的名帶有些許該物質或獸特有的屬性（例如火、堅韌、疾速、靈敏、記憶等），成為**帶屬性的詞元**，即後續附魔、強化、成長的根源素材。**邊界**：同類指人與人；對人剝名時 token 沒反應，無法成立。人對獸、人對物剝名可行（陽光奇點，不走黑暗設計風）。

### 二、詞元的分類與層級（豐富設定）

- **鎂字（通用賦權詞元）**：唯一僅適瑞式族人血脈賦權的詞元；作為世界通用貨幣與賦權基準，各國共用、瑞式族人壟斷賦權故永遠中立。不從剝名取得，而由機構與流通賦權。
- **剝名所得詞元（帶屬性的名）**：從物或獸剝下的名，帶有該來源的屬性（火、冰、堅韌、疾速、靈敏等），可入竅穴、可寫入裝備、可參與自創技能。依來源稀有度與強度，可視為地域級或更高。
- **通用詞元**：人人可感應、可紀錄、可納入內盤；對應常見物產名、基礎技藝名、通用概念。流通廣，權重較低，多用作表達與紀錄的磚石，亦可入竅產生基礎型特殊加成。
- **地域／傳承詞元**：特定地域、族群、師門才有的詞元；對應稀有物產、地方傳說、門派心法名。蒐集能解鎖地域敘事、任務或技能樹分支；入竅或入裝備時往往觸發較強特殊加成或連線。
- **禁忌／失傳詞元**：曾從世界詞盤上被抹除或封存的名；對應禁術、已滅絕之物等。持有或使用可能帶來強大效果，也可能觸發代價；本專案陽光向，可設定為稀少但不走殘酷代價。
- **個體專屬詞元**：極少數個體在機緣下與某種「名」共振，產出僅此人可用的詞元；與**自創技能**直接掛鉤——你創的技，你的名，你的詞元。

### 三、詞盤與個體（深度對接）

- **內（詞）盤與竅穴**：詞盤在人身稱為**內（詞）盤**。詞元以**竅穴**為載體——每個竅穴即詞元在身體上的落點。**竅穴打通**即產生**普通加成**（例如體質、氣脈、基礎抗性等數值或狀態）；**詞元入竅**後則在該竅產生**特殊加成**（依詞元屬性：火竅抗火與火系增幅、疾速竅行動加成等）。多個詞元竅穴依規則形成**連線**（如經脈拓樸、星圖式連線等），連線帶來額外加成（套裝效果、共鳴、技能前置等）。打通／入詞／連線與加成均由**後端計算認證**，作為數值與玩法依據。
- **個體即「行走的詞盤」**：每個角色自帶內盤與有限竅穴；先打通竅穴得普通加成，再將蒐集或剝名所得的詞元入竅得特殊加成，最後透過連線放大效果。個體強度＝**盤的深度與廣度**＋**連線帶來的加成**——能叫出多少名、能對多少事物賦權、竅穴如何連線，皆可量化與認證。
- **精神念紋**：除瑞式族人外，其他族裔也可能帶有不同性質的念紋（偏戰鬥、工藝、契約等）。念紋決定與哪類詞元「共振」更容易，即**天賦方向**；瑞式族人念紋獨一無二在「僅能對鎂字賦權」，故入各國金融機構、永遠中立。
- **成人禮與入盤**：瑞式族人成人禮後正式入機構詞盤；一般人則可視為第一次把內盤與世界詞盤接軌——註冊冒險者、入籍某城、或習得第一個可被紀錄的**自創技能**。
- **個體養成迴圈**：探索與戰鬥 → **剝名**或蒐集詞元 → 以鎂交易或取得資源 → **打通竅穴**（普通加成）→ **詞元入竅**（特殊加成）→ **連線**（後端認證加成）→ 詞元排列成**自創技能**、剝下的名寫入**裝備附魔／強化**。成長即內盤擴充與竅穴連線的深化；後端統一計算並認證所有加成結果。

### 四、詞盤與自創技能（深度對接）

- **技能即「可被執行的名」**：一個技能即一組詞元依特定順序與條件被喚起、賦權、執行的結果。詞元可來自剝名（帶屬性）、流通的鎂、或傳承登錄的名。**自創技能**即「把內盤上的詞元（含剝下的名）排列成世界尚未登錄的新名」——造句後經**詞盤祭煉成敘事**，系統判定語意通過即生成效果。
- **自創技能的誕生**：內盤具備足夠且相容的詞元（含剝名所得）時，造句／排列成敘事 → **祭煉**於詞盤 → **系統判定**中文邏輯、語順、語意，合則生成屬性加成或技能效果；或在實戰、修行中觸發「命名瞬間」由念紋與內盤承載。消耗剝下的名、鎂或兩者。
- **系統命名**：技能名稱由**系統萃取簡名**（2～4 字），不開放玩家自訂。描述寫完整敘事＋發明者＋效果。例：敘事「我以火擊之」→ 系統授予「火球術」（單體火傷）；「我以烈火連擊之」→ 連擊或小範圍；「我會跑」→ speed+1.5％ 等強化。
- **登錄與流通**：自創技能可僅止於內盤（獨門）；亦可交付代價申請「登錄」到工會、門派或世界詞盤邊緣，成為他人可學習或購買的「名」，形成**技能樹與流派**。多敘事可組**陣列**，對應高階效果。
- **詞元組合與強度**：技能強度、範圍、代價與詞元數量、稀有度、是否含鎂或特殊詞元有關；高階自創技能往往需要地域詞元、剝名所得或個體專屬詞元。與竅穴連線、裝備附魔共用同一套詞元經濟，邏輯閉環。

---

### 五、詞盤與裝備附魔強化（深度對接）

- **附魔的根源＝剝下的名**：從物或獸剝下的「名」帶有該來源的屬性；**附魔**即把這些帶屬性的詞元，透過儀式、鎂與念紋，寫入裝備的詞盤（附魔槽），使裝備獲得對應權能（鋒利、堅韌、火炎、疾速等）。物或獸已消散，名可流轉——附魔師與冒險者用的，正是剝名所得。
- **附魔與強化**：裝備即「外在詞盤」，具有限槽位。鑲嵌剝下的名＝一次賦權；**強化**＝以更多鎂或同名／同系詞元對同一附魔名做二次賦權，提升強度或疊加效果。**強化階**對應賦權深度；過載可能導致槽位損毀或附魔降階（陽光向可設為可修復）。
- **與內盤的共鳴**：裝備附魔詞元與穿戴者內盤（竅穴中的詞元、念紋）相容時，可觸發**共鳴**，產生額外效果或減耗；不相容則可能折損。個體成長與裝備養成共用同一套詞元與鎂的取得與消耗，形成閉環。

---

### 六、賦權的兩面與瑞式族人的軸心

- **賦權的兩面**：一為「賦予名分」——使某物、某約、某筆帳被承認、可紀錄；一為「賦予權能」——持鎂者能交換、訂約、執行技能、驅動附魔。瑞式族人專司鎂字賦權，故金融、契約與鎂的流轉皆依賴他們；永遠中立，各國敢共用同一套鎂。
- **若無瑞式族人**：鎂字無法賦權，鎂不能流通、不能作為強化與自創技能的通用消耗與交易媒介；經濟軸與信任軸崩潰，內盤與裝備詞盤仍可運作，但退回到以物易物或地域代幣。

### 七、邏輯閉環總覽

- **世界觀閉環**：token 無處不在、使人知悉物質的名 → 名由詞元構成、可被紀錄為文字 → 人類能剝名（僅對物與獸）、剝下的名帶屬性 → 名／詞元可流轉入內盤（竅穴、連線）、裝備（附魔槽）、技能（造句→祭煉成敘事→系統判定語意→授名）→ 鎂字為名之名的單位、經瑞式族人賦權後通用 → 各國共用鎂、瑞式永遠中立 → 表達、交易、賦權、成長皆在「名」的體系內完成，萬事萬物受 token 影響一以貫之。
- **資源與玩法閉環**：探索／戰鬥 → **剝名**或**蒐集詞元** → 詞元入**內盤**（竅穴打通→入詞→連線，後端認證加成）→ **造句／祭煉成敘事** → 系統判定語意 → **自創技能**（系統授名、登錄流通）或**陣列**；剝下的名另可寫入**裝備附魔與強化**（槽位、強化階、共鳴）→ 鎂經瑞式賦權用於**交易、強化、技能消耗** → 個體與裝備變強 → 再探索。三者（個體、技能、裝備）共用同一套詞元與鎂的取得、消耗與賦權邏輯。
- **設計邊界（陽光奇點）**：人對人剝名 token 無反應；禁忌詞元可稀少但不走殘酷代價；附魔過載可設為可修復。整體風格閉環於「陽光」定位。
- **閉環自檢**：詞元來源＝蒐集＋剝名；詞元去向＝內盤（竅穴／連線）＋造句／祭煉（技能／陣列）＋裝備附魔／強化；鎂＝僅瑞式賦權、通用媒介；名＝詞元構成、剝名對象、祭煉／登錄內容。以上無斷點。

| 詞元入竅 | 將詞元納入已打通之竅穴，產生特殊加成（依詞元屬性）；多竅連線後加成由後端認證 |
| 普通加成 | 僅打通竅穴即得之加成，不依詞元種類 |
| 特殊加成 | 詞元入竅後依該詞元屬性產生之加成；連線可再放大 |
| 詞元竅穴／連線 | 多個詞元竅穴依規則（如經脈拓樸）連線後產生各類加成；後端計算認證 |
| 詞盤容量 | 個體內盤或裝備可承載的詞元上限，可隨成長或品質擴充 |
| 念紋 | 先天或後天與某類詞元共振的印記；瑞式為鎂字賦權專用 |
| 命名瞬間 | 自創技能誕生契機：詞元排列成新名並被內盤接納 |
| 剝名 | 因 token 賦權，將物或獸上的「名」剝下；物／獸消散，剝下的名帶屬性，供附魔、強化、成長。人對人剝名 token 無反應；人對獸、對物可行。 |
| 登錄 | 把名（技能、契約、所有權）寫入機構或世界詞盤，使之可流通或可查 |
| 附魔槽／詞元槽 | 裝備上可鑲嵌詞元（剝下的名等）的位置，即該裝備詞盤容量 |
| 強化階 | 同一附魔名被重複賦權的深度，對應附魔強度 |
| 共鳴 | 裝備附魔詞元與穿戴者內盤（竅穴詞元、念紋）相容時的加成或減耗 |

## 遊戲大數據檢視（與整體設計對齊）

以下從專案整體設計關鍵字與關鍵字評估（DESIGN_KEYWORDS、KEYWORDS_ASSESSMENT）檢視詞盤系統是否閉環、對齊、有無缺口。

| 維度 | 整體設計要點 | 詞盤對齊狀況 |
|------|--------------|--------------|
| **經濟** | 真實經濟；評估推薦：單一通用貨幣＋產消閉環、時間綁定產出 | **鎂**＝世界通用貨幣、僅瑞式賦權，可視為單一貨幣之世界觀包裝。產消閉環需在細規補齊：鎂的產出（任務／交易／報酬）與回收（稅／維護／強化消耗）由後端唯一真理建模。 |
| **自創技能** | 符合世界規則；主管決斷：組合模組＋配方／藍圖＋後端驗證＋工具熟練度延伸（ABCE） | **造句→祭煉→系統判定→系統授名**＝組合模組（詞元排列）＋後端驗證（語意判定）＋不開放自訂名（系統萃取 2～4 字）。配方／藍圖＝登錄於世界詞盤之敘事可被學習。工具熟練度可疊加於竅穴／連線或裝備熟練度。 |
| **裝備** | 裝備＝工具；全身多部位；工具熟練度 | 裝備＝**外在詞盤**；附魔槽、強化階、共鳴與「工具」語義一致；剝下的名寫入裝備＝權能來源有據，**不可無中生有**。 |
| **不可無中生有／永序守衡** | 思維邏輯閉環；不可僅憑文字創造物件 | 詞元來源＝**蒐集＋剝名**（剝名則物／獸消散）；技能＝造句＋系統判定通過才授名；鎂由機構賦權、不從剝名取得。名與詞元皆有其來源與去向，無斷點。 |
| **後端唯一真理** | 數據、量化後端計算；前端轉文字敘述 | 竅穴打通／入詞／連線、祭煉、系統判定語意、系統授名、附魔／強化階，**均由後端計算認證**；詞盤邏輯可全在後端，前端僅呈現結果與敘事。 |
| **玩家＝NPC** | 同一套規則與身份 | 用語：token＝玩家**（包含 NPC）**滿世界蒐集的物件；詞元、內盤、剝名、造句可對 NPC 同一套邏輯，萬名 NPC 若有詞盤則與玩家共用同一套詞元經濟與判定。 |
| **社會／門派／城鎮** | 低中高巨型社會；門派、氏族、城鎮、國家 | 登錄可寫入**工會、門派、世界詞盤**；機構詞盤由瑞式執掌、各國國有金融機構；與社會層級、門派傳承、城鎮經濟可對接。 |
| **戰鬥** | 全自動、文字 log；輸贏靠裝備與技能等後端屬性 | 技能＝可執行的名；強度與詞元組合、竅穴連線、裝備附魔／共鳴掛鉤；戰鬥結果由後端依詞盤與裝備數據計算，與「田忌賽馬」一致。 |

**結論（大數據）**：詞盤系統與現行關鍵字及評估決斷**對齊良好**，可視為自創技能、個體成長、裝備附魔強化的**具體化與世界觀包裝**。建議在細規中補齊：**(1) 鎂的產消閉環表**、(2) **NPC 詞盤／蒐集行為**與觀測坍縮的介面（可選）、(3) **系統判定語意**的規則邊界（中文邏輯、語順、允許詞元集合），以利後端唯一真理實作。
